# mDrop

A lightweight macOS file drop utility. Launch from command line, drop files into the window to stage them, then drag out to other apps or locations.

## Install

```bash
brew install m1nts02/tap/mdrop
```

## Usage

### Launch

```bash
# Launch directly
mdrop

# Launch with files
mdrop /path/to/file1 /path/to/file2
```
